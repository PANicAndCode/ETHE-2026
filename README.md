Eaton's Easter Treasure Hunt

This package keeps the same QR code values and the same team order as the original hunt, but uses Eaton-specific riddles.

Setup
1. Put your Supabase URL and anon key in supabase-config.js
2. Run eatons_setup.sql in Supabase
3. Upload the site files to a separate repo/site
4. Hard refresh phones after uploading

Note: The map image files are placeholders copied from the other hunt. Replace them or turn the map off if you do not want to use it.
